# Minesweeper

javaで開発したマインスイーパーです。

コンソール版を作成した後、swingを使用し簡易的なGUI版へ移植しました。 

訓練校で学習した二次元配列、二重for文への理解向上、オブジェクト指向を意識したクラス設計や、画面更新の仕組みを学ぶ事を目的とし製作しました。

## 🎮 プレイ方法

### コンソール版

1. 行番号を入力
2. 列番号を入力
3. 地雷を避けながらすべてのマスを開く
4. 入力方法を選び、oを入力すると盤面を開き、fを入力すると地雷があると思った場所に旗（印）を置ける

### GUI版

左クリック：マスを開く

## 🎯 Game Rules
- 盤面上の地雷を避けるゲーム
- 数字は周囲8マスの地雷数
- 0は連鎖的に自動オープン
- 地雷を踏むとゲームオーバー
